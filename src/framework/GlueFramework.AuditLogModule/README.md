# Glue Framework Audit Log Module

OrchardCore module providing audit logging capabilities.

## Features

- Automatic audit event recording via `[Audit]` attribute
- Audit log admin UI
- Audit event filtering and search
- Integration with Glue Framework Core

## Admin Endpoints

- `GET /Admin/AuditLogAdmin` - View audit logs
- Audit log cleanup functionality

## Configuration

Audit logging is enabled by default when the module is active.
